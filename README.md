# 🚀 Mission Control Skeleton
### The Premium Operational Core for Autonomous Agent Orchestration

**Mission Control Skeleton** is a sanitized, "Plug & Play" distribution of the highly advanced Agent HQ orchestration hub. Designed for commercial release, it provides a sophisticated, white-labelable interface for managing a fleet of autonomous agents, their skills, and their strategic missions.

![Agent OS](https://img.shields.io/badge/Architecture-Agent_OS-38bdf8?style=for-the-badge)

## ✨ Key Features

- **🛡️ Sanitized Core**: Stripped of personal credentials and agent data; ready for immediate deployment by third parties.
- **🧙 Setup Wizard**: A premium, multi-step onboarding experience for injecting API keys (Anthropic, OpenAI, ElevenLabs) upon first launch.
- **🛠️ Universal Skills Hub**: A dynamic registry for attaching and toggling capabilities (Browsing, Trading, Filesystem) across the fleet.
- **📱 Unified Shell**: A high-end, responsive UI optimized for mobile (iOS/Android) and desktop, featuring smooth glassmorphism and Framer Motion transitions.
- **🔒 Security-First**: Integrated MFA, JWT-based authentication, and a local-first data model (using `openclaw_data`).
- **🏠 Local/Cloud Hybrid**: Seamlessly toggle between cloud-based intelligence and local model execution (Ollama/DeepSeek).

## 🚀 Quick Start

1. **Clone the repository**:
   ```bash
   git clone https://github.com/nakedeyent-art/MissionControlSkeleton.git
   cd MissionControlSkeleton
   ```

2. **Install Dependencies**:
   ```bash
   npm install
   ```

3. **Launch the Core**:
   ```bash
   npm run dev
   ```

4. **Initialize**: Follow the **Setup Wizard** at `http://localhost:18789` to input your API keys and define your master admin persona.

## 🏗️ Technical Stack

- **Frontend**: React, Vite, Framer Motion, Lucide Icons.
- **Backend**: Node.js, Express, Axios.
- **Mobile**: Capacitor (iOS/Android ready).
- **Data**: Local JSON/SQLite storage via the OpenClaw protocol.
